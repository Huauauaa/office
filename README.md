## prepare

- `python3 -m venv venv`
- `. venv/bin/activate`

### Database

#### init db

- `flask db init`

#### migrate

- `flask db migrate`

#### create table

- `flask db upgrade`

## Install

- `pip install -r requirements.txt`
- `pnpm install`

## Build

- `pnpm run build`

## Dev run

- `flask run -p 8888`
- `pnpm run dev`
